How to View Your Birthday Page

A simple guide to accessing Mandathi's birthday page from GitHub

Step-by-Step Instructions

1

Go to your GitHub repository

Navigate to your GitHub repository named "my-web-link"

2

Find the bday.html file

Locate the bday.html file in your repository file list

3

Click on the file

Click on the bday.html file to view its content on GitHub

4

View the raw file

Click the "Raw" button to see the raw HTML file

5

Copy the URL

Copy the URL from your browser's address bar

6

Paste into a new tab

Paste the URL into a new browser tab and replace "github.com" with "htmlpreview.github.io"
